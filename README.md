# Next Move Micro-frontend

## Install

```bash
git clone --recursive https://github.com/jungai/next-move-micro-frontend-workspace.git

make update
```

## Tools
- single-spa
- vue 3 (vue-cli)
- spectrum web component
- system-js
- vue eco lib(e.g. vue-router, vueUse, vee-validate)
